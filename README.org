* About

This course teaches modern software development practices for scientific research, with a focus on leveraging AI coding agents (Claude Code, GitHub Copilot, Codex, Gemini CLI, etc.) as collaborative tools. Students learn to build, test, document, and distribute Python packages while developing critical skills in evaluating and directing AI-generated code.

* Topics

| Week | Topics                                             |
|------+----------------------------------------------------|
| 1    | AI-Assisted Development, Terminals and Shells      |
| 2    | Git and Version Control, Working with APIs         |
| 3    | Python Packaging, Testing with AI                  |
| 4    | Code Quality, Debugging and Profiling with AI      |
| 5    | GitHub Collaboration, CI/CD with GitHub Actions    |
| 6    | Documentation, Configuring and Extending AI Agents |
| 7    | Project Presentations                              |

Optional lectures cover Python classes, ethics/copyright/licensing, web apps, security, databases, CLI tools, profiling/optimization, uv (modern Python package management), release lifecycle (versioning, changelogs, PyPI publishing), scientific computing pitfalls (floating-point, units, validation), the human side of AI-assisted development (burnout, skill atrophy, sustainable practices), and case studies in AI-assisted development.
