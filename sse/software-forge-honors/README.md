# Software Forge Honors

A recognition program for demonstrating mastery of scientific software development. Complete badges to earn ranks and show your expertise as a Software Craftsperson.

---

## Program Overview

**Mission:** Recognize students who go beyond the basics to demonstrate true understanding of software development concepts and their application to scientific computing.

**Timeline:** Designed to be achievable within one semester

**Philosophy:** Like a craftsperson earning their credentials at a forge, each badge and rank represents demonstrated skill in building, testing, and maintaining quality software.

---

## Quick Start

1. **Complete Lectures & Quizzes** - Work through course material
2. **Earn Badges** - Demonstrate mastery in each area (see [Badges](badges.md))
3. **Achieve Ranks** - Combine badges with capstone activities (see [Ranks](ranks.md))
4. **Track Progress** - Use the tracking system to monitor your journey

## The Four Ranks

| Rank | Requirements | Meaning |
|------|--------------|---------|
| **Code Apprentice** | 1 badge | Beginning the journey |
| **Software Crafter** | 2 badges + midterm | Building competence |
| **Software Engineer** | 4 badges + field experience | Professional level |
| **Master Craftsperson** | All 5 badges + final + project | Full mastery |

---

## How It Works

### Earning Badges

Each badge requires:
1. **Quiz Performance**: Average 75%+ on section quizzes
2. **Assignments**: Complete coding assignments
3. **Badge Activity**: One hands-on activity demonstrating the skill

### Advancing Ranks

Each rank requires:
1. **Badge Count**: Accumulate required badges
2. **Exam Performance**: Meet exam thresholds
3. **Capstone Activity**: Rank-specific challenge

---

## Recognition

Completed achievements are recognized through:
- **Digital Badges**: Shareable credentials
- **Course Records**: Official documentation
- **Portfolio Building**: Each badge adds to your software portfolio
- **GitHub Profile**: Real contributions and projects
