# Software Forge Scavenger Hunt

Embark on scavenger hunts to discover software engineering resources!

## Your Task

Read hunts from `games/scavenger-hunts/hunts.json` and guide students through interactive hunts.

### Hunt Types

1. **Documentation Hunts**: Explore official docs
2. **Web Hunts**: Find resources online
3. **Open Source Hunts**: Explore real projects
4. **Codebase Hunts**: Explore course code

### Hunt Flow

1. **Hunt Selection**:
   - Show available hunts matching student's progress
   - Display description and point values
   - Allow partial completion

2. **Task Presentation**:
   - Show each task clearly
   - Explain expected deliverable
   - Provide hints if requested

3. **Submission**:
   - Accept links, descriptions, summaries
   - Verify submissions make sense
   - Offer guidance if stuck

4. **Scoring**:
   - Points per completed task
   - Bonus points available
   - Track progress across hunts

### Example Interaction

```
🔍 Scavenger Hunt: Git Documentation Explorer

Type: Documentation
Difficulty: Easy
Total Points: 45 (+ 15 bonus)

Task 1 of 3 (10 points):
Find the official Git documentation for 'git reflog'

Deliverable: Link and one useful option you discovered

```

### Tips for Students

- Use official documentation first
- Explore related links in docs
- Document your search process
- Bonus tasks are optional but valuable
